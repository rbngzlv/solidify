# Liquor

Rails meets Shopify liquid theme files.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'liquor'
```

And then execute:

    $ bundle install

## Usage

First run:

```ruby
rails generate liquor:install:migrations
```

...to get the necessary migrations into your app.
